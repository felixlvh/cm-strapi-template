# Content Metric (CM) Project

This is a Content Metric application powered by Strapi. Use the CLI commands below to develop, build, and run your project.

### `develop`

Start your CM application with autoReload enabled.

```
npm run develop
```

### `start`

Start your CM application with autoReload disabled (production mode).

```
npm run start
```

### `build`

Build the admin panel.

```
npm run build
```

## Deployment

Your CM project is managed and deployed through the Content Metric control plane. Use the dashboard to start, stop, and monitor your project.
